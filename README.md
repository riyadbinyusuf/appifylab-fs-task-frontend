This is an Appifylab Fullstack task's Frontend repo. Built with Next JS, SWR, React Boostrap, Zod.

Installation:

1. Install dependencies: `npm install`
2. Duplicate env.example, rename it to .env.local, and update the base API URL
3. Start the server: `npm run dev`

Features:

- User registration and login
- Post create with text and image
- Posts infinite scroll pagination
- Post like and comment
- Comment create with text and image
- Comment like and reply

Usage:

- Clone the repository
- Install dependencies
- Set up the environment variables
- Start the server

Live demo:

- Backend: [Appifylab Fullstack task backend API](https://appifylab-fs-task-backend.onrender.com)
- Frontend: [Appifylab Fullstack task frontend](https://appifylab-fs-task-frontend.vercel.app)
